# YouTube Ad Time Tracker

YouTubeの広告で無駄にされた時間を可視化するChrome拡張機能

## 📋 概要

YouTubeで動画を視聴する際に表示される広告によって失われた時間を、広告ごとに集計・可視化します。単純接触効果による無意識的な広告への好感度向上を防ぐことを目的としています。

## ✨ 主な機能

- ✅ YouTube動画視聴中の広告表示時間を自動計測
- ✅ 広告ごとの累計表示時間を集計
- ✅ 広告の視聴回数をカウント
- ✅ 広告主別の時間統計を表示
- ✅ 総広告時間・広告種類数・総視聴回数の表示
- ✅ 時間順・回数順でのソート機能

## 🚀 インストール方法

### 1. リポジトリのクローン

```bash
git clone https://github.com/yourusername/ad-hate-checker.git
cd ad-hate-checker
```

### 2. アイコンの生成（オプション）

既にアイコンが生成されていますが、再生成する場合:

```bash
pip install pillow
python3 create_icons.py
```

### 3. Chrome拡張機能として読み込み

1. Chromeブラウザを開く
2. アドレスバーに `chrome://extensions/` と入力してEnter
3. 右上の「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `ad-hate-checker/extension` フォルダを選択

## 📖 使い方

### 基本的な使い方

1. 拡張機能をインストール後、YouTubeで動画を視聴
2. 広告が表示されると自動的に時間の計測が開始されます
3. ブラウザのツールバーにある拡張機能アイコンをクリック
4. ポップアップで広告ごとの統計情報を確認

### 統計画面の見方

#### サマリー情報
- **総広告時間**: すべての広告で失われた時間の合計
- **広告の種類**: 記録された異なる広告の数
- **総視聴回数**: 広告を視聴した総回数

#### 広告リスト
各広告について以下の情報を表示:
- 広告主名
- 広告タイトル
- 累計表示時間
- 視聴回数
- 平均表示時間
- 初回・最終視聴日

#### ソート機能
- **時間順**: 累計表示時間の長い順に並べ替え
- **回数順**: 視聴回数の多い順に並べ替え

#### データ削除
「データ削除」ボタンで全ての記録データをリセットできます。

## 🔧 技術仕様

- **Manifest Version**: 3（最新のChrome拡張仕様）
- **対応ブラウザ**: Google Chrome（Manifest V3対応バージョン）
- **必要な権限**:
  - `storage`: 広告データの保存
  - `https://www.youtube.com/*`: YouTubeページへのアクセス

## 💾 データ保存

すべてのデータはローカル（Chrome Storage API）に保存されます。外部サーバーへのデータ送信は一切行いません。

### 保存されるデータ

1. **広告履歴** (`adHistory`)
   - 各広告の表示時間
   - タイムスタンプ
   - 広告主・タイトル情報

2. **広告統計** (`adStats`)
   - 広告ごとの累計時間
   - 視聴回数
   - 初回・最終視聴日時

## 🎯 目的と効果

### 単純接触効果とは
人は繰り返し接触するものに対して好意を持つ傾向があります（ザイアンスの法則）。広告業界はこの心理効果を利用しています。

### この拡張機能の効果
- 広告によって失われた時間を可視化することで、広告への否定的な認識を強化
- 単純接触による無意識の好感度向上を抑制
- 広告の「コスト」を時間という形で明確化

## 🔒 プライバシー

- すべてのデータはローカルストレージに保存
- 外部サーバーへの通信なし
- 個人情報の収集なし
- YouTubeの閲覧履歴は記録しません（広告情報のみ）

## ⚠️ 注意事項

- この拡張機能は広告をブロックするものではありません
- 広告の表示時間を計測・記録するのみです
- YouTube Premiumなど、広告非表示サービスを利用している場合は機能しません
